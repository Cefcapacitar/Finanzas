#Finanzas

Este repositorio contiene fragmentos de código en Python aplicados a las finanzas, desarrollados con fines estrictamente pedagógicos.
El material está estructurado para que los estudiantes puedan copiar y pegar las secciones de código en sus propios Jupyter Notebooks o entornos de Google Colab. El objetivo es facilitar la experimentación directa con modelos financieros, análisis de datos y algoritmos de evaluación de manera práctica y dinámica.
